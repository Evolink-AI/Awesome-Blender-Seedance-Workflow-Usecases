# Security Policy

This repository primarily contains documentation, structured source data, media references, and maintenance scripts. Please do not disclose credentials, private source exports, unpublished media, or personal information in issues or pull requests.

## Reporting A Vulnerability

Use [GitHub private vulnerability reporting](https://github.com/Evolink-AI/Awesome-Blender-Seedance-Workflow-Usecases/security/advisories/new) for security-sensitive reports. Include the affected file or workflow, reproduction details, and potential impact.

For ordinary content corrections, broken links, attribution issues, or use case submissions, use the public issue templates instead.
